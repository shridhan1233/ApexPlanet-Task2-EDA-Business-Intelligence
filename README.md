# Task 2: Exploratory Data Analysis (EDA) & Business Intelligence

## Objective

The objective of this task is to perform Exploratory Data Analysis (EDA) on a sales dataset and generate meaningful business insights using data visualization, SQL queries, and business intelligence techniques.

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* SQLite
* Google Colab

## Task Components

* Data Overview
* Descriptive Statistics
* Univariate Analysis
* SQL Business Questions
* Multivariate Analysis
* Static Dashboard Mock-up
* Business Insights

## Key Findings

* Identified top-performing products and categories.
* Analyzed customer purchasing behavior.
* Examined city-wise sales performance.
* Performed correlation analysis.
* Designed a business dashboard.

## Outcome

Successfully applied data analytics and business intelligence techniques to transform raw sales data into actionable business insights.
